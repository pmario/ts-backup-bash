# ts-backup-bash

After [downloading](https://github.com/pmario/ts-backup-bash/releases/) the script, make sure it is executable.

Prerequisites:

 - curl
 - sed
 - printf
 

Executable:

The ts-backup file needs to be an executable!

```
chmod +x ts-backup.sh
```

Usage: 

```
./ts-backup.sh <tsUserName>

eg: 

./ts-backup.sh pmario
```
